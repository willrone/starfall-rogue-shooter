## 四、隐私政策

```html
<h3>隐私政策</h3>
<p>本游戏尊重并保护所有用户的隐私。本政策适用于<b>星坠幸存者</b>（以下简称"本游戏"）。</p>
<h4>1. 信息收集</h4>
<p>本游戏不收集用户的个人身份信息。游戏运行期间可能收集以下非个人信息：</p>
<ul>
  <li>设备型号、操作系统版本（用于性能优化）</li>
  <li>游戏内的匿名统计数据（击杀数、波次、通关率）</li>
</ul>
<h4>2. 广告服务</h4>
<p>本游戏包含激励视频广告，由抖音广告平台提供。广告SDK可能收集设备标识符用于广告投放优化，详情见抖音隐私政策。</p>
<h4>3. 数据存储</h4>
<p>游戏进度和设置存储在本机设备上，不会上传至服务器。</p>
<h4>4. 儿童隐私</h4>
<p>本游戏适合12岁以上用户。我们不会故意收集儿童的个人信息。</p>
<h4>5. 联系我们</h4>
<p>如有隐私相关问题，请联系开发者邮箱。</p>
```
