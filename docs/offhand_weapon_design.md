# 副武器系统设计 · 基线 v1.0

> 定稿日期：2026-07-02 | 状态：✅ 已开发（代码已实现，与设计一致）

---

## 设计原则

1. **主武器 = DPS输出**，现有 17 把照旧
2. **副武器 = 协同修短板**，不切换、不瞄准，自动运行
3. **独立升级面板**，每把 5 级（T1-T5）
4. **独立材料**：合成+升级用专用材料，跟主武器 Boss 材料不冲突
5. **装备栏**：1 主武器 + 1 副武器同时装备，战斗中同时生效

## 副武器材料设计

### 材料类型

| 材料名 | 获得方式 | 用途 |
|--------|---------|------|
| 净化结晶 | 通关波10获得1个 | 合成1-5号副武器 |
| 虚空碎片 | 通关波15获得1个 | 合成6-10号副武器 |
| 时间晶格 | 通关波20获得1个 | 合成11-15号副武器 |
| 通用金粉 | 任何Boss战获得1个 | 升级副武器T1→T2 |

### 合成公式

```
合成「回旋利刃」   = 净化结晶 ×1 + 合金 50
合成「影刃猎手」   = 虚空碎片 ×1 + 合金 65
合成「纳米修复器」 = 时间晶格 ×1 + 合金 80
...
```

### 升级公式

```
T1 → T2 = 通用金粉 ×3 + 合金 100
T2 → T3 = 通用金粉 ×5 + 合金 180
T3 → T4 = 通用金粉 ×8 + 合金 280
T4 → T5 = 通用金粉 ×12 + 合金 400
```

---

## 装备规则

- 战斗内同时装备 1 主武器 + 1 副武器
- 副武器不占用主武器装备槽
- 战斗外（机库）可选择/切换副武器
- 副武器独立升级，不依赖主武器等级
