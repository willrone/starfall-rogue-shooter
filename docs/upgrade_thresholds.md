**分析维度**：子弹数 = ceil(敌人HP / 单发伤害)
**公式**：`dmg = wd×(1+(lv-1)×0.12) + 16×0.15 + ap_upgrade`  
**ap_upgrade** = +5/级（升级专属属性，attackPower）  
**HP缩放** = baseHP × (1 + wave×0.028 + wave×45×0.0018)

- 波1-3（早期）：每升1-2级，子弹数少1发
- 波4-6（中期）：子弹数维持稳定（2-3发），升级提供手感提升
- 波7+（后期）：子弹数缓慢增加，升级对抗HP膨胀

**问题标记**：
- `❌` = 台阶太晚（Lv6+才触发，波1-5体验断层）
- `⚠` = 台阶太早（波1就一击秒，升级无感知）
- `✓` = 台阶合理

| 武器 | 早期台阶(波1-5) | 中期台阶(波6-10) | 晚期台阶 | 综合评价 |
|------|----------------|-----------------|---------|---------|
