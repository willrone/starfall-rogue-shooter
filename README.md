# 星坠幸存者

Cocos Creator 3.8.8 平面射击肉鸽原型。

## 当前玩法

- 玩家只控制移动，武器会自动瞄准最近的怪物并开火。
- 战斗改为无尽出击：没有倒计时胜利，玩家死亡或主动撤离才会结算。
- Boss 被击杀后不会结束战斗，而是进入下一轮怪潮循环，怪物密度、血量和伤害持续提高。
- 战斗内击杀怪物掉落经验，角色升级后从 3 个临时强化中选择 1 个。
- 怪物会随时间快速增密，包含小怪、冲刺怪、坦克、分裂怪、精英和后期首领。
- 当前怪物目录为 55 种普通怪物变体 + 1 个首领，共 56 种。
- 当前武器目录为 20 个武器家族 × 10 个型号，共 200 把武器，另有 5 件通用装备。
- 每次出击按存活时间、击杀数和 Boss 击杀轮数结算资源；主动撤离保留全部奖励，死亡会折损部分时间奖励。
- 永久资源包含合金、核心、装备碎片、生体样本、电路板、虚空晶体。
- 装备升级会消耗不同材料；锁定装备可以用装备碎片和对应材料合成。
- 进入游戏会先到机库主页，玩家可以查看仓库、翻页浏览装备、查看已拥有装备属性，并配置同时生效的出战槽。
- 出战槽分为 4 个武器槽 + 2 个装备槽；槽内装备都会在战斗中生效。
- 机库里可以用永久资源升级装备属性，每次显示 8 个可升级或待解锁条目，升级和出战配置都会保存到本地存档。
- 战斗中会显示怪物受伤数字、玩家受击掉血数字和回血数字。

## 占位美术

- 透明 PNG 素材在 `assets/resources/art/placeholder/`。
- `contact_sheet.png` 是整套占位素材总览。
- `tools/generate_placeholder_art.py` 可以重新生成同风格素材。
- 游戏会优先从 `resources` 加载这些素材；加载失败时回退到脚本绘制的几何图形。

## 操作

- 桌面端：WASD 或方向键移动。
- 触屏端：按住并拖动任意战斗区域移动。
- 射击、选目标、开火全部自动完成。
