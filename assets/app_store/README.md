# 抖音上传素材（草案）

这些素材由 `tools/generate_ui_assets.py` 基于当前游戏内主角、怪物、武器资源合成，保持统一的暖色卡通科幻风格。

## 文件

```text
app_icon_1024.png              # 小游戏应用图标草案，1024×1024
share_cover_1280x720.png       # 分享/宣传横图草案
loading_key_art_720x1280.png   # 竖版加载页关键视觉草案
```

## 后续注意

- `app_icon_1024.png` 可作为开放平台上传图标候选，但提交前建议在抖音开发者后台预览小尺寸可读性。
- 审核截图应优先来自真机/开发者工具实际游戏画面，不建议用合成图替代。
- 若后续正式隐私政策、适龄提示、版面文案变更，分享图和加载图也应同步更新。
