## 当前文件

```text
app_icon_1024.png              1024×1024 应用图标候选
share_cover_1280x720.png       1280×720 分享/宣传横图候选
loading_key_art_720x1280.png   720×1280 竖版加载视觉候选
```

这些文件用于素材归档和提交候选，不会因为放在 `assets/app_store/` 就自动进入游戏或自动同步到抖音开放平台。`submission/` 中的正式提交文件也是独立交付物，不能假设与这里的候选图始终相同。

## 提交时实时核对

平台尺寸、格式、透明通道、文件大小、文案、安全区、适龄和审核要求会变化。每次提审必须以当时抖音开放平台后台和开发者工具显示的最新要求为准，逐项核对：

- 上传后的裁切区域和小尺寸识别度
- 是否允许透明通道以及实际背景色
- 横竖版素材比例、文件格式和大小限制
- 游戏名称、版本文案、适龄和隐私信息是否与当前提交一致
- 审核截图是否来自当前真机/开发者工具版本，而不是历史合成图

`tools/generate_ui_assets.py` 当前仍引用缺失或过期的角色/Boss 尺寸，不能作为权威的一键重生成入口；其状态见 [美术资源替换指南](../../docs/ART_REPLACEMENT_GUIDE.md) 和 [基线差异清单](../../docs/BASELINE_GAPS.md)。
